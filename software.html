R packages

I am the maintainer of the following R packages:

  • games (coauthored with Curt Signorino), which contains functions to estimate and analyze models of interdependent strategic behavior. The functionality of the package is described and illustrated in detail in our Journal of Statistical Software article "Estimation of Extensive Form Games in R".
  • polywog (coauthored with Curt Signorino), which allows users to estimate relationships of unknown functional form using bootstrapped basis regression. The package also provides routines to plot fitted values and marginal effects from fitted models. The software and the underlying statistical techniques are described in our working paper "Bootstrapped Basis Regression with Variable Selection: A New Method for Flexible Functional Form Estimation".
  • pbivnorm, which provides a vectorized function to compute the bivariate normal CDF. pbivnorm is based on code from the mnormt package.
